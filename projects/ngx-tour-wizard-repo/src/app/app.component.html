Install NGX Tour Wizard

  • npm i ngx-tour-wizard

I'm an anchor

You can have an anchor here, but set the target over there: useful to target hidden elements (or not).

My target for step 2

Step 3 will appear with a 2 seconds delay, useful to handle async calls during a tour, for example.

You can also delay the appearance of the next step: pass a Subject into the subjectForNext or subjectForPrev property of the step config.
The step will be shown after the Subject is triggered!
You can customize the appearance of any single popper, by using the style property in the step config!

Features

  • Uses Popper 2 which has been proven to be a very effective and performing library for tooltips and popovers
  • Optional Keyboard navigation: use the arrows to move between steps and the ESC key to exit the tour!
  • Automatic scroll to the step reference
  • Accepts HTML and can even be highly connected with your components, by passing a component ref in the customPopperContent property.
